CSS: Cascading Style Sheets

Uma introdução completa

O que é CSS?

  • Folhas de estilo em cascata
  • Usado para estilizar elementos HTML
  • Separa estrutura (HTML) da apresentação (CSS)

Tipos de CSS

  • Inline: direto no elemento
    <h1 style="color:red">Oi</h1>
  • Interno: dentro da tag <style> no HTML
  • Externo: em um arquivo .css separado
<p style="color: red;">Texto inline</p>

Seletores CSS

  • Elementos: p, h1
  • Classes: .botao
  • IDs: #cabecalho
  • Atributos: [type="text"]
  • Pseudo-classes: :hover, :first-child

Box Model

Content → Padding → Border → Margin

Propriedades comuns

  • color, background-color
  • font-size, font-family
  • width, height
  • margin, padding, border
  • display, position

Pseudo-classes: :hover

button:hover {
  background-color: green;
  color: white;
}

Interação ao passar o mouse

Transitions

div {
  transition: background-color 0.3s ease;
}
div:hover {
  background-color: #f00;
}

Transições suaves entre estados

Animations

@keyframes mover {
  from { transform: translateX(0); }
  to { transform: translateX(100px); }
}

.box {
  animation: mover 2s infinite alternate;
}

Criação de movimentos animados

Flexbox

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

Layout flexível e responsivo

Grid

.container {
  display: grid;
  grid-template-columns: 1fr 2fr;
  gap: 10px;
}

Layout em grade com controle total

Media Queries

@media (max-width: 768px) {
  body {
    font-size: 14px;
  }
}

Estilos adaptativos para diferentes telas

Custom Properties (Variáveis)

:root {
  --cor-primaria: #3498db;
}
button {
  background-color: var(--cor-primaria);
}

Reuso e manutenção facilitada

Frameworks CSS

  • Bootstrap
  • Tailwind CSS
  • Bulma

Facilitam o desenvolvimento com componentes prontos

Boas práticas

  • Evitar !important
  • Reutilizar classes
  • Organizar por componentes
  • Usar variáveis e utilitários

Exemplo completo

.botao {
  background-color: #3498db;
  color: white;
  padding: 10px 20px;
  border: none;
  transition: background 0.3s;
}
.botao:hover {
  background-color: #2980b9;
}

Resumo

  • CSS deixa seu site bonito e organizado
  • Você pode mudar cores, fontes, tamanhos e muito mais
  • Existem várias formas de aplicar CSS
  • Pratique para aprender cada vez mais!
