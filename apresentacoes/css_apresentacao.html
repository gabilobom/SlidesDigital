CSS:
Cascading Style Sheets

O que é CSS?

  • Folhas de estilo em cascata
  • Usado para estilizar elementos HTML
  • Separa estrutura (HTML) da apresentação (CSS)

Tipos de CSS

  • Inline: direto no elemento
    <h1 style="color:red">Oi</h1>
  • Interno: dentro da tag <style> no HTML
  • Externo: em um arquivo .css separado
<p style="color: red;">Texto inline</p>

<!-- Interno -->
<style>
  p { color: blue; }
</style>
<p>Texto interno</p>

<!-- Externo -->
<link rel="stylesheet" href="estilos.css">
<p>Texto externo</p>

Seletores CSS

  • Elementos: p, h1
  • Classes: .botao
  • IDs: #cabecalho
  • Atributos: [type="text"]
  • Pseudo-classes: :hover, :first-child

Box Model

Box Model define como o espaço de cada elemento é calculado:
Content (conteúdo) → Padding (preenchimento)
Border (borda) → Margin (margem externa).
Isso permite controlar espaçamento, bordas e o layout dos elementos na página.

Tamanhos em CSS

  • px — pixels (valor fixo, não responsivo)
  • % — porcentagem em relação ao elemento pai
  • em/rem — relativo ao tamanho da fonte
  • vw/vh — porcentagem da largura/altura da tela
div {
  width: 50%;   /* metade do elemento pai */
  height: 200px; /* altura fixa */
}

h1 {
  font-size: 2em; /* 2x o tamanho da fonte do elemento pai */
}

section {
  width: 80vw; /* 80% da largura da tela */
}

Use % para layouts flexíveis e responsivos!

Propriedades comuns

/* Cor do texto e fundo */
color: red;
background-color: #f8f8f5;

/* Fonte */
font-size: 20px;
font-family: Arial, sans-serif;

/* Tamanho */
width: 200px;
height: 100px;

/* Espaçamento */
margin: 10px;
padding: 20px;
border: 1px solid #222;

/* Layout */
display: flex;
position: absolute;

/* Sombra e borda arredondada */
box-shadow: 0 2px 8px #0003;
border-radius: 8px;

/* Opacidade e ordem */
opacity: 0.8;
z-index: 10;

/* Overflow e alinhamento de texto */
overflow: hidden;
text-align: center;

/* Cursor e transição */
cursor: pointer;
transition: background 0.3s;

Pseudo-classes: :hover

button:hover {
  background-color: green;
  color: white;
}

Interação ao passar o mouse

Transitions

div {
  transition: background-color 0.3s ease;
}
div:hover {
  background-color: #f00;
}

Transições suaves entre estados

Animations 

@keyframes mover {
  from { transform: translateX(0); }
  to { transform: translateX(100px); }
}

.box {
  animation: mover 2s infinite alternate;
}

Criação de movimentos animados

Flexbox

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

Layout flexível e responsivo

Grid

.container {
  display: grid;
  grid-template-columns: 1fr 2fr;
  gap: 10px;
}

Layout em grade com controle total

Media Queries

@media (max-width: 768px) {
  body {
    font-size: 14px;
  }
}

Estilos adaptativos para diferentes telas

Custom Properties (Variáveis)

:root {
  --cor-primaria: #3498db;
}
button {
  background-color: var(--cor-primaria);
}

Reuso e manutenção facilitada

Frameworks CSS

  • Bootstrap
  • Tailwind CSS
  • Bulma

Facilitam o desenvolvimento com componentes prontos

Boas práticas

  • Evitar !important
  • Reutilizar classes
  • Organizar por componentes
  • Usar variáveis e utilitários

Exemplo completo

.botao {
  background-color: #3498db;
  color: white;
  padding: 10px 20px;
  border: none;
  transition: background 0.3s;
}
.botao:hover {
  background-color: #2980b9;
}
